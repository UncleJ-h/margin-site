# MARGIN Website

MARGIN 产品官网 - 日程收集箱应用。

## 页面结构

```
margin-site/
├── index.html          # 落地页（中文）
├── index-en.html       # 落地页（英文）
├── privacy.html        # 隐私政策（中文）
├── privacy-en.html     # 隐私政策（英文）
├── terms.html          # 服务条款（中文）
├── terms-en.html       # 服务条款（英文）
├── support.html        # 支持/FAQ（中文）
├── support-en.html     # 支持/FAQ（英文）
└── README.md           # 本文件
```

## 技术栈

- 纯 HTML5 + CSS3
- 零 JavaScript，零构建工具
- GitHub Pages 托管

## 设计规范

- 背景色：`#faf9f7`（暖纸色）
- 文字色：`#1a1a1a`
- 辅助色：`#d97706`（琥珀）
- 字体：SF Pro Text / -apple-system

## 部署

1. 推送到 GitHub 仓库
2. Settings → Pages → Source: Deploy from a branch
3. Branch: main / root
4. 等待部署完成
