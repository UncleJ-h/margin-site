隐私政策

最后更新：2026 年 1 月 20 日

你的数据留在你的设备上。
我们不追踪你，不卖你的数据。

概述

MARGIN 是一款日程收集箱应用，帮助你把截图和文本变成日历事件。本隐私政策说明我们如何收集、使用和保护你的信息。

我们收集什么

1. 你分享的内容

当你通过分享菜单将截图或文本分享给 MARGIN 时：

  • 我们处理什么：你明确分享的图片和文本
  • 如何处理：发送至 AI 服务提供商（OpenRouter）进行事件提取
  • 提取什么：事件标题、日期和时间
  • 本地存储什么：仅在你设备上存储提取的事件信息
  • 不存储什么：原始图片和文本处理后立即删除

2. 日历数据

A) 添加事件（默认）

  • 我们访问什么：仅在你确认时写入日历（iOS 17+ 权限模型）
  • 我们写入什么：仅你明确确认的事件
  • 我们不做什么：永远不会未经确认自动添加事件

B) 日历消化（可选）

如果你在设置中启用了「日历消化」功能：

  • 我们访问什么：完整日历访问权限（用于读取事件）
  • 我们读取什么：你的日历事件，用于生成日程结构摘要
  • 如何处理：在你的设备上本地处理；日历数据不会上传到我们的服务器
  • 我们不做什么：不会编辑或删除现有日历事件

3. 订阅信息

  • Apple 处理什么：所有支付由 Apple 通过 StoreKit 处理
  • 我们本地存储什么：你的订阅状态（档位、到期日期）
  • 我们不存储什么：支付方式、信用卡信息、账单详情

4. 设备信息

  • 我们收集什么：设备语言区域（用于语言选择）、iOS 版本（用于兼容性）
  • 我们不收集什么：设备标识符、位置数据、通讯录、浏览历史

数据使用方式

数据类型 用途 存储位置
分享的内容 提取事件信息 通过 API 处理，不保留
提取的事件 显示待确认候选 仅在你的设备
确认的事件 添加到日历 iOS 日历
日历消化数据 生成日程摘要 仅在你的设备
订阅状态 启用功能 仅在你的设备

第三方服务

OpenRouter API

  • 用途：AI 驱动的图片和文本事件提取
  • 发送数据：你分享的截图和文本
  • 数据保留：实时处理，不存储

Apple 服务

  • StoreKit：处理所有订阅支付
  • EventKit：写入确认的事件；仅在启用日历消化时本地读取事件
  • App Store：分发应用

数据安全

  • HTTPS 加密：所有 API 通信使用 TLS 1.3
  • 本地存储：使用 iOS 安全存储
  • 无追踪：无分析 SDK，无用户追踪
  • 最小收集：只收集必要数据

儿童隐私

MARGIN 分级为 4+，不会故意收集 13 岁以下儿童的信息。应用不含任何不适合儿童的内容。

政策变更

我们可能会更新本隐私政策。更新时：「最后更新」日期会变更，重大变更会在应用更新中说明。

简而言之

  • 不卖你的数据
  • 不追踪你
  • 数据存储在你的设备
  • 日历消化在本地处理，不上传
  • 删除应用后所有本地数据一并删除
