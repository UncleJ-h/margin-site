2. Service Description

MARGIN is a calendar inbox application that helps you turn screenshots and text into calendar events. The App provides the following features:

  • Receive screenshots and text via Share Sheet
  • AI-powered event recognition (title, date, time)
  • Pending event list management
  • One-tap confirmation to add to system calendar
  • Daily briefing (view today's schedule summary)
  • Pro features: Voice briefing, weekly/monthly analysis
